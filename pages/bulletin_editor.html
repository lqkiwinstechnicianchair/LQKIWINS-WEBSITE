BULLETIN EDITOR

A Bulletin Editor is responsible for making the flyers for the event pages as well as any graphics the club may need. In the case that the club does not have a Technology Chair, the Bulletin Editor must also create and run the club's website and create other digital media such as videos. The Bulletin Editor is also a part of the Creative Committee of the club, and so they also must be able to get along with all the members and help out whenever needed.
